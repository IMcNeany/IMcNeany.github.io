University Projects

Procedural Animation

For my masters dissertation I looked into procedural animation techniques, focusing on the Jacobian transverse, FABRIK algorithm and physics-based motion. The final demo includes an editor to allow for customised walking patterns to be created. Unable to link until after final marks have been published(Nov 2020).

Don't Walk by

A group project produced from scratch in Unity. The game was created to raise awareness of hazards within the workplace. Within the project I implemented all the animations using spline within the project, creating a system to generate randomly looking characters as well a a system for destroying tiles. A Trailer is available here. Unable to link github project as owned by the University.

Track Attack

This project was a group project. We had the task of taking a game another group had made and to improve on it. The game initally was 4x local player, battle with carts on rails. We took the original concept and changed it to work with networked AR. I implemented the AR along with the non AR fallback version.

Project code here.

Factory Tycoon

With this project I got the opportunity to work for a client at the University of Bath. The project is a management game that will help raise awareness of a sustainable buisness. The player is put in charge of a packaging factory and needs to balance social, environmental, economical values. Project code here.

Road Data

This was a group project that involved creating a simulation from a set of data. The data chosen was pedestrian accidents on a road in 2018. The data included time of day, weather, and lighting conditions. I took the raw data and created a system to read it in from the excel file format. The data was then set up to be used to calculate chance precentages. The project is avalible

here.
